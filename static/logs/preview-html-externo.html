Preview html desde otro dispositivo

Obtener la dirección y generamos un qr para hacer mas fácil el paso de información de la computadora al teléfono 

import socket
import pyqrcode

port = "8080"
s = socket.socket(socket.AF_INET, socket.SOCK_DGRAM)
s.connect(("8.8.8.8", 80))
a = s.getsockname()[0]
b = "https://" + a + ":" + port

print(pyqrcode.create(b).terminal())
print(b)

s.close()

y luego creamos el servidor de prueba 

python3 -m http.server 8080
